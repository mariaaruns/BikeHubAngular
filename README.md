## ✨ Overview

BikeHub is a full-stack web application built with **Angular 21** on the frontend and **.NET 9 Minimal API** on the backend. It provides a seamless experience for bike enthusiasts to browse, manage, and interact with bike-related content.

---

## 🚀 Getting Started

### Prerequisites

- Node.js (LTS recommended)
- Angular CLI v21+
- .NET 9 SDK *(for backend)*

### Installation

```bash
# Clone the repository
git clone https://github.com/mariaaruns/BikeHub.git
cd BikeHub

# Install dependencies
npm install

# Start the development server
ng serve
```

Open your browser and navigate to **[http://localhost:4200](http://localhost:4200)**. The app hot-reloads on file changes.

## 🛠 Development

### Generate Components

```bash
# Generate a new component
ng generate component component-name

# See all available schematics
ng generate --help
```

### Build for Production

```bash
ng build
```

Compiled output is stored in the `dist/` directory, optimized for performance and speed.

## 🧪 Testing

### Unit Tests

Powered by **[Vitest](https://vitest.dev/)** — fast, modern, and Jest-compatible.

```bash
ng test
```

### End-to-End Tests

```bash
ng e2e
```

> Angular CLI does not include an e2e framework by default — choose one that suits your workflow (e.g., Playwright, Cypress).

## 🔌 Backend API

The frontend consumes a **.NET 9 Minimal API** backend.

**Clone the backend:**
```bash
git clone https://github.com/mariaaruns/BikeHub.git
```
